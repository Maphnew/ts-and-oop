# tsc config

- Watch mode

```
tsc logging.ts -w
```

- Create tsconfig

```
tsc --init
```

- Watch mode for all

```
tsc -w
```

## outDir

- Specify an output folder for all emitted files.

## rootDir

- Specify the root folder within your source files

- exclude
- include
